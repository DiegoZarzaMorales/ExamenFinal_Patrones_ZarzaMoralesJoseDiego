# ExamenUnidad4y5

Alumno: Diego Zarza Morales

Proyecto: Sistema de Entrega por Drones (Patrón Singleton + Patrón Object Pool + Patrón Estado + Patrón Fachada + Arquitectura por Capas)

Descripción breve:
Este proyecto implementa un sistema completo de entregas utilizando una flota de drones repartidores. La arquitectura combina 5 patrones de diseño: Singleton para la central de control única, Object Pool para la gestión eficiente de 5 drones reutilizables, Estado para manejar los estados de carga de cada dron, Fachada para simplificar la orquestación de servicios, y Capas para separar responsabilidades en 4 capas (Dominio, Negocio, Infraestructura y Presentación). El sistema incluye persistencia en archivos, validaciones en tiempo real, estadísticas de entregas y simulación de fallos cada 5 entregas.
